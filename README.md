# Credit Card Fraud Detection

## Overview

This project uses Machine Learning to detect fraudulent credit card transactions.

## Technologies

- Python
- Pandas
- Scikit-Learn
- Streamlit
- Random Forest

## Github Link
https://github.com/nithika1405/credit-card-fraud-detection

## Dataset

Fraud Detection Dataset

Files:
- fraudTrain.csv
- fraudTest.csv

## Features

- Data Preprocessing
- Label Encoding
- Random Forest Classifier
- Fraud Prediction
- Interactive Streamlit UI

## Accuracy

~99% Accuracy

## Run

python src/train_model.py

streamlit run app.py
